Progressive Web Application (PWA) Webpage


Characteristics of Progressive Web Application


Offline possibility:

service workers cache the necessary content for the PWA application and serve it afterward.

This feature makes PWA offline-capable as users can still interact with the application even offline.

The offline access was previously possible with Application Cache.


Installable:

PWA can be promoted by the browser for installation on the device.

After installed, PWA appears to users as a platform-specific app.

A feature of their device they can launch directly from the operating system.
